# Active Learning (능동 학습)

## 1. 개요 (Overview)
Active Learning(능동 학습)은 모델이 학습 성능을 높이기 위해 가장 필요한 데이터를 스스로 선별하여 사람(Oracle)에게 레이블링(정답 표시)을 요청하는 기법입니다.

## 2. 왜 필요한가?
딥러닝 모델을 학습시키려면 레이블이 달린 데이터가 많이 필요한데, 데이터를 일일이 레이블링하는 비용(시간, 돈)이 매우 큽니다. 능동 학습은 최소한의 데이터만 레이블링하여 최대한의 성능을 내는 것을 목표로 합니다.

## 3. 작동 과정 (Process)
1.  적은 양의 레이블된 데이터로 초기 모델을 학습시킵니다.
2.  레이블이 없는 대량의 데이터(Unlabeled Pool) 중 모델이 가장 헷갈려하는(Uncertainty) 데이터를 고릅니다.
3.  사람에게 그 데이터의 레이블을 물어봅니다(Query).
4.  레이블을 얻은 데이터를 학습 데이터에 추가하고 모델을 다시 학습시킵니다.
5.  목표 성능에 도달할 때까지 반복합니다.

## 4. 샘플링 전략 (Query Strategy)
- **Uncertainty Sampling**: 모델의 예측 확률이 0.5에 가까운(가장 불확실한) 데이터를 선택합니다.
- **Diversity Sampling**: 데이터 공간에서 골고루 퍼져 있는 데이터를 선택하여 다양성을 확보합니다.
