# Sentiment Analysis (감성 분석)

## 1. 개요 (Overview)
Sentiment Analysis(감성 분석)는 텍스트에 담긴 작성자의 감정, 태도, 의견 등을 파악하는 [자연어 처리(NLP)](./NLP.md) 기술입니다. 주로 긍정(Positive), 부정(Negative), 중립(Neutral)으로 분류합니다.

## 2. 주요 접근법
- **Lexicon-based (사전 기반)**: 감성 어휘 사전(예: 좋다=+1, 싫다=-1)을 이용하여 점수를 계산합니다.
- **Machine Learning**: 나이브 베이즈, SVM 등의 알고리즘으로 텍스트를 분류합니다.
- **Deep Learning**: [RNN](./RNN.md), [LSTM](./RNN.md), [BERT](./BERT.md) 등을 이용하여 문맥을 파악하고 더 정확하게 분류합니다.

## 3. 응용 분야
- 소셜 미디어 모니터링 (브랜드 평판 분석)
- 영화/제품 리뷰 분석
- 고객 피드백 분석
