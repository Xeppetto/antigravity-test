# Federated Learning (연합 학습)

## 1. 개요 (Overview)
Federated Learning(연합 학습)은 데이터를 중앙 서버로 모으지 않고, 각 사용자의 기기(Edge Device)에서 모델을 학습시킨 후, 학습된 결과(가중치 업데이트)만을 서버로 보내 합치는 분산 학습 기술입니다.

## 2. 등장 배경
개인정보 보호(Privacy)에 대한 중요성이 커지면서, 민감한 데이터(의료 기록, 키보드 입력 등)를 서버로 전송하지 않고도 AI 모델을 학습시킬 방법이 필요했습니다.

## 3. 작동 과정 (Process)
1.  서버가 글로벌 모델을 각 기기에 배포합니다.
2.  각 기기는 자신의 로컬 데이터로 모델을 학습시킵니다.
3.  각 기기는 학습된 모델의 업데이트 정보(Gradient 등)만을 서버로 전송합니다. (데이터는 기기에 남음)
4.  서버는 수신한 업데이트들을 평균(Aggregation)내어 글로벌 모델을 개선합니다.
5.  개선된 글로벌 모델을 다시 기기들에 배포하고 이 과정을 반복합니다.

## 4. 장점
- **Privacy**: 원본 데이터가 기기 밖으로 나가지 않아 개인정보가 보호됩니다.
- **Efficiency**: 네트워크 대역폭을 절약할 수 있습니다.
